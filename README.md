##Anleitung:
```
    Zur Nutzung der App muss auf der Startseite ein Passwort eingegeben und auf 'Login' geklickt werden.
    Daraufhin wird man auf die Hauptseite weitergeleitet. Hier kann man sich per Mausklick in verschiedenen Kategorien seine Passwörter anzeigen lassen.
    Über ein Klick auf 'Passwort hinzufügen' kann ein neues Passwort angelegt werden.
    Man wird auf eine neue Seite geführt, auf der man alle Felder ausfüllen und anschließend auf 'Bestätigen' klicken muss.
    Es wurde ein neues Passwort angelegt, falls man es sich währenddessen noch anders überlegt kann man den Vorgang über ein Klicken auf 'Zurück' abbrechen.
    Über ein Klick auf 'Passwort löschen' kann ein Passwort gelöscht werden.
    Man wird auf eine neue Seite geführt, auf der man über das Klicken auf den 'Löschen' Button ein Passwort löschen kann.
    Das Passwort wurde gelöscht, falls man es sich währenddessen noch anders überlegt kann man den Vorgang über ein Klicken auf 'Zurück' abbrechen.
    Über das Klicken des 'Abmelden' Buttons auf der Hauptseite gelangt man wieder auf den Anmelde-Startbildschirm.
    
    Die Passwörter werden über eine symmetrische AES-Verschlüsselung verschlüsselt in der Datenbank gespeichert.
```
    

##ProjektIdee:
```
    Einfacher Passwort-Manager
    -> Passwörter können eingespeichert werden 
    -> Passwörter werden als verschlüsselt angezeigt 
    (erst bei Aufforderung/durch Aktion wird das Passwort in Klarschrift angezeigt)
    -> als dynamische Funktion z.B. Dropdown Menü
    ggf. Zusatz:
    -> Sortierung der Webseiten nach Alphabet
```
